# Calculadora de juros compostos

FORMULA :

    Montante Total :
         MT = C * ( 1 + T )**p
    
    Total Investido :
        TInv = C - M * P
    
    Total de Juros :
        TJr = MT - C + M * P

MT = montante
C = capital aplicado
T = taxa de juros composto
p = periodo
J = juros composto
